# 创作人格量表在线测试

## 项目概述

这是一个基于"创作人格"理论的在线心理量表测试网站，旨在帮助创作者了解自己在创作过程中呈现的人格轮廓。该理论框架由雪背乌鸦提出，复合月亮进行理论拓展并设计量表，海蛞蝓提供理论修改意见。注意，对应测试的主要功能是侧写而不是证明。

**访问地址**：[https://rauuuuuuch.github.io/creativepersonality.github.io/](https://rauuuuuuch.github.io/creativepersonality.github.io/)

## 理论基础

### 基本预设
1. **创作人格而非真实人格**：描述的是创作中呈现的人格特征，而非日常生活中的真实人格。
2. **流动但相对稳定**：创作人格是流动的，但在特定创作阶段（如一张专辑、一个写作阶段）内可视为相对稳定的基调。
3. **适用范围**：适用于以个人表达为核心的创作行为（音乐、文学、视觉艺术、电影），不太适用于高度功能性的创作（商业设计、技术文档）。

### 理论框架
任何创作行为至少涉及四个要素：**作者** (Author)、**受众** (Audience)、**作品** (Work) 和 **素材** (Material)。此外，作者的意识始终面对**真实世界** (Reality) 的底层结构。

以作者为中心节点，向外辐射出四条关系轴，五个维度锚定在这些关系轴上：

| 维度 | 关系轴 | 连续体/结构 |
|------|--------|-------------|
| **姿态** (Posture) | 作者-受众 | 低姿态 ↔ 高姿态 |
| **心理年龄** (Psychological Age) | 作者-真实世界 | 社会化 ↔ 非社会化 |
| **认知模式** (Cognitive Mode) | 作者-作品 (质地) | 思维 / 感觉 / 幻觉 (三极) |
| **表达距离** (Expressive Distance) | 作者-作品 (方式) | 真诚 / 迂回 / 游离 (三极) |
| **朝向** (Orientation) | 作者-素材 | 内指 ↔ 外指 |

## 量表结构

### 题目信息
- **题目数量**：48题
- **评分方式**：5点李克特量表 (1=完全不符合, 2=比较不符合, 3=不确定, 4=比较符合, 5=完全符合)
- **答题时间**：约10-15分钟

### 维度与子维度
1. **姿态** (8题)
   - 高姿态 (4题)
   - 低姿态 (4题)

2. **心理年龄** (10题)
   - 社会化/成人 (4题)
   - 前社会化/孩童 (3题)
   - 反社会化/青春期 (3题)

3. **认知模式** (12题)
   - 思维 (4题)
   - 感觉 (4题)
   - 幻觉 (4题)

4. **表达距离** (10题)
   - 真诚 (4题)
   - 迂回 (3题)
   - 游离 (3题)

5. **朝向** (8题)
   - 内指 (4题)
   - 外指 (4题)

## 计分方法

### 原始分计算
每个子维度的题目得分相加，得到原始分数。

### 标准化百分比
由于各子维度题目数量不同，原始分需转换为0-100%的标准化百分比：

```
标准化百分比 = (原始分 - 题目数量) / (题目数量 × 4) × 100%
```

**解释**：
- 所有题目选1分 = 0%
- 所有题目选3分（中立）= 50%
- 所有题目选5分 = 100%

### 判定规则

#### 双极维度（姿态、朝向）
使用统一阈值 T = 12%：
- **高姿态**：高姿态% > 低姿态% + 12%
- **低姿态**：低姿态% > 高姿态% + 12%
- **中间姿态**：差值 ≤ 12%
- **朝向**同理

#### 三极维度（心理年龄、认知模式、表达距离）
将三个百分比从高到低排序：
- **单一主导类型**：最高分 > 第二高分 + 12%
- **混合类型**：差值 ≤ 12%（标注为"类型A/类型B"）

## 结果展示

测试完成后，系统将生成：

1. **总体标签**：五个维度的判定结果串联
   - 示例：`高姿态·社会化（成人）·感觉-幻觉·真诚·内指`

2. **详细分析**：每个维度包含
   - 维度名称和关系轴标注
   - 各子维度的条形图（可视化百分比）
   - 基于判定结果的解读文字

3. **15种可能结果解读**：
   - 姿态：高姿态、低姿态、中间姿态
   - 心理年龄：社会化、前社会化、反社会化得分最高时
   - 认知模式：思维、感觉、幻觉得分最高时
   - 表达距离：真诚、迂回、游离得分最高时
   - 朝向：内指、外指、内指-外指均衡
